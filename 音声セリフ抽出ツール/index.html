音声セリフ抽出ツール

音声ファイルからセリフ区間を検出し、タイムスタンプ付きで一覧表示します

🎙️
音声ファイルをドロップまたはクリックして選択
WAV / MP3 / OGG / M4A / FLAC 対応
-40dB
💡 音声認識はブラウザの Web Speech API を使用します（Chrome推奨）。初回のみマイク許可を求められます。認識精度はブラウザ・環境に依存します。
⚠️ マイク許可ポップアップが毎回表示される場合：ローカルファイル（file://）で開いている場合、ブラウザは許可を記憶できません。ローカルサーバー経由（例：python -m http.server 8000）で http://localhost:8000 からアクセスすると許可が永続化されます。
