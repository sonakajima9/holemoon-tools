# HoleMoon Tools

HoleMoonプロジェクトで開発したツール群を公開しているリポジトリです。  
すべてのツールはブラウザで動作するシングルファイルのWebアプリです。インターネット接続なしでも使用できます。

## ツール一覧

| ツール名 | 概要 |
|---------|------|
| [CSVジェネレーター](./CSVジェネレーター/) | 音声収録用CSVファイルを生成するツール |
| [VoiceSync](./VoiceSync/) | 台本と音声ファイルをリンクする同期ツール |
| [文字変換ソフト](./文字変換ソフト/) | テキスト変換・置換ツール |
| [音声CSVパターンツール](./音声CSVパターンツール/) | 音声収録パターンのCSV生成ツール |
| [音声セリフ抽出ツール](./音声セリフ抽出ツール/) | 台本からセリフを抽出するツール |
| [ElevenLabs辞書](./ElevenLabs辞書/) | ElevenLabs用PLS発音辞書 |

## 使い方

1. 使用したいツールのフォルダを開く
2. `index.html` をダウンロードしてブラウザで開く（オフライン動作可）
3. または [GitHub Pages](#) からオンラインで使用する

## 更新について

このリポジトリは [HoleMoon](https://github.com/sonakajima9/HoleMoon) の `ツール管理用フォルダ/` と GitHub Actions で自動同期されています。  
HoleMoon 側でツールが追加・更新されると、自動的にこのリポジトリにも反映されます。
